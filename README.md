# CIS7-Vigenere-Cipher
CIS7 project implementing the Vigenère Cipher for encryption and decryption using C++.

## Author
Omar Tourjoman  

## Project Case
Case 3 – Vigenère Cipher Decryption

## Description
This project demonstrates how the Vigenère Cipher works by encrypting and decrypting alphabetic messages using a repeating keyword and modular arithmetic.

## How to Run the Program
1. Open the `src` folder.
2. Compile and run the `project_case3.cpp` file using any C++ compiler.
3. When prompted:
   - Enter a plaintext message.
   - Enter a keyword.
4. The program will display:
   - Generated key  
   - Encrypted text  
   - Decrypted text
